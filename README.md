# AssemblyExperiment

汇编语言课程实验项目仓库，面向 8086/DOS 环境，使用 MASM、LINK、DEBUG 和 DOSBox 完成实验代码编写、编译、运行与调试。

当前仓库主要整理了前两次实验的相关材料：

- 实验一：熟悉 `debug` 使用
- 实验二：顺序程序设计

## 项目结构

```text
AssemblyExperiment/
├─ ex1/
│  └─ 1.doc            # 实验一实验报告
├─ ex2/
│  ├─ 2_1.asm          # 字符串输入与回显程序
│  ├─ 2_2.asm          # Y = 2X + 3 计算程序
│  └─ 2.doc            # 实验二实验报告
├─ 2026汇编语言实验指导书.docx
├─ 汇编语言实验报告格式2026.doc
└─ README.md
```

## 实验环境

根据实验指导书，本项目默认使用以下环境：

- `DOSBox`：模拟 DOS 运行环境
- `MASM`：汇编
- `LINK`：链接
- `DEBUG`：调试

说明：

- 源码采用 16 位 MASM 语法，基于 `int 21h` DOS 中断完成输入输出。
- 建议在 DOSBox 中挂载当前项目目录后进行编译和运行。

## 快速开始

### 1. 挂载项目目录

在 DOSBox 中执行：

```bat
mount d D:\000-Students\wcy_gwh\AssemblyExperiment
d:
```

### 2. 编译并运行实验二程序

进入 `ex2` 目录后，可按下面方式操作。

编译运行 `2_1.asm`：

```bat
cd ex2
masm 2_1.asm;
link 2_1.obj;
2_1.exe
```

编译运行 `2_2.asm`：

```bat
masm 2_2.asm;
link 2_2.obj;
2_2.exe
```

如果需要调试，可继续执行：

```bat
debug 2_1.exe
debug 2_2.exe
```

## 已实现内容

### 实验一：熟悉 `debug` 使用

`ex1` 目录当前保存的是实验报告：

- `1.doc`：实验一报告文档

该实验对应指导书中的“熟悉 debug 使用”，重点是掌握：

- `debug` 的基本命令和使用流程
- 程序装载、单步执行与寄存器/内存观察
- DOS 功能调用的基本调试方法

### 实验二：顺序程序设计

`ex2` 目录包含两个已完成的汇编程序和对应实验报告。

#### 1. `2_1.asm`：字符串输入与回显

功能：

- 使用 `int 21h` 的 `0Ah` 功能输入字符串
- 将输入缓冲区末尾转换为 `$`
- 使用 `09h` 功能输出字符串

程序特点：

- 展示了 DOS 缓冲区输入格式的使用方法
- 适合配合 `debug` 观察缓冲区内容变化
- 对应指导书中“输入一个字符串并输出”的实验内容

#### 2. `2_2.asm`：`Y = 2X + 3`

功能：

- 从键盘读取一位十进制数 `X`
- 计算 `Y = 2X + 3`
- 在下一行输出提示信息和十进制结果

程序特点：

- 使用 `mul` 完成乘法
- 使用 `aam`、`aaa` 进行十进制调整
- 演示了字符输入、ASCII 转数字、结果显示的完整流程

#### 3. `2.doc`

- 实验二实验报告文档
