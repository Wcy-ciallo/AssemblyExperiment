# AssemblyExperiment

汇编语言课程实验项目仓库，面向 8086/DOS 环境，当前已整理实验一到实验四的源码、目标文件、列表文件和实验报告，并补充了 `hw5` 目录下的第五次作业源码。

├─ ex2/

│  └─ 3.docx          # 实验三实验报告
├─ ex4/
│  ├─ 4_1.asm         # XLAT 查表
│  ├─ 4_2_1.asm       # 直接/寄存器间接寻址查成绩

## 构建说明

仓库中的源码采用 16 位 MASM 语法，程序主要通过 `int 21h` 完成输入输出与退出。

如果需要重新生成可执行文件，通常使用以下流程：

```bat
masm xxx.asm;
link xxx.obj;
xxx.exe
```

调试时可按需要使用：

```bat
debug xxx.exe
```

## 已整理实验

### 实验一

`ex1` 目录当前保存实验报告：

- `1.docx`

### 实验二

`ex2` 目录包含两个顺序程序设计示例和实验报告。

`2_1.asm`

- 使用 `int 21h` 的 `0Ah` 功能输入字符串
- 将输入缓冲区改写为 `$` 结尾
- 使用 `09h` 功能完成字符串回显

`2_2.asm`

- 从键盘读取一位十进制数 `X`
- 计算 `Y = 2X + 3`
- 输出计算结果

`2.docx`

- 实验二实验报告

### 实验三

`ex3` 目录主要围绕数组首地址、偏移量和内存访问展开。

`3_1.asm`

- 通过 `offset arr` 获取数组首地址
- 使用 `[bx-2]`、`[bx+14]` 这类相对位移形式访问内存
- 体现变量与数组在数据段中的连续存储关系

`3_2.asm`

- 以 `score` 数组为基础计算不同元素地址
- 使用 `bx`、`si`、`di`、`bp`、`dx` 保存不同偏移位置
- 展示按字单元数组进行地址推导与访问的方法

`3.docx`

- 实验三实验报告

### 实验四

`ex4` 目录主要整理了查表和多种寻址方式示例。

`4_1.asm`

- 使用 `XLAT` 指令按索引在字符表中查值

`4_2_1.asm`

- 根据班级号和学号定位成绩
- 使用直接寻址与寄存器间接寻址访问二维成绩表

`4_2_2.asm`

- 使用基址变址寻址访问成绩表

`4_2_3.asm`

- 使用相对基址变址寻址访问成绩表

`4_3.asm`

- 建立学生姓名指针表
- 根据输入的学号索引并输出对应姓名

`4.docx`

- 实验四实验报告

### 作业五

`hw5` 目录整理了第五次作业中的条件分支、数组统计、字符串输入和位操作题目。

`5_26.asm`

- 读取字节变量 `x`
- 通过 `shr` 和进位标志判断奇偶
- 输出 `Y` 或 `N`

`5_27.asm`

- 根据 `x` 的正负和是否为零选择不同分支
- 分段计算变量 `y`

`5_28.asm`

- 当 `x < 0` 时求 `|x|`
- 当 `x >= 0` 时计算 `z = 4x + y/16`

`5_29.asm`

- 顺序扫描字数组 `M`
- 分别统计正数、负数和零的个数

`5_30.asm`

- 逐字符读入字符串到缓冲区
- 在遇到空格或达到长度上限时结束输入
- 将结果改写为 `$` 结尾并输出

`5_31.asm`

- 对寄存器 `AX` 的高低字节执行按位与和移位
- 提取 `AH`、`AL` 中指定的低位字段
