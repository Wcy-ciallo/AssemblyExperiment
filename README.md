# AssemblyExperiment

8086/DOS 汇编语言课程实验仓库。源码采用 16 位 MASM 语法，主要在 DOSBox 环境中通过 `masm`、`link`、`debug` 完成编译、链接、运行和调试。

当前已整理实验一到实验八的源码、报告、流程图/运行结果截图，并保留了 `hw5`、`hw6`、`hw7` 目录下的作业源码。

## 构建与运行

单个 `.asm` 文件按 MASM 传统流程编译：

```bat
masm xxx.asm;
link xxx.obj;
xxx.exe
```

需要观察寄存器、内存或定位逻辑错误时使用：

```bat
debug xxx.exe
```

仓库没有统一构建脚本，每个实验源码单独编译。

## 实验说明

### 实验五：流程图设计以及逻辑错误调试

指导书要求掌握标准流程图绘制、结构化程序设计方法，并在 DOSBox、MASM、LINK、DEBUG 环境中完成调试。

`lab05/5_1.asm` 完成 0-255 十进制数到十六进制数的转换：

- 使用 `int 21h/09h` 输出 `DEC=` 和 `HEX=` 提示。
- 使用 `int 21h/01h` 逐字符读入十进制数字，回车结束。
- 按 `num = num * 10 + digit` 累加十进制值。
- 反复除以 16，余数 0-9 加 `30h`，10-15 加 `37h`，得到十六进制字符。
- 余数先按低位到高位存入 `hex` 数组，输出时逆序打印。
- `lab05/lab05.png` 保存了对应流程图。

`lab05/test.asm` 是 0Ah 缓冲输入测试程序，`TEST.LST` 是其 MASM 列表文件。

### 实验六：分支程序设计

指导书要求掌握无条件转移、条件转移、`cmp`、`test`、位操作指令，并把已有程序整理成至少三个功能的菜单程序。

`lab06/6_1.asm`

- 根据实验五流程图实现十进制到十六进制转换。
- 代码逻辑与 `lab05/5_1.asm` 基本一致，可作为流程图到汇编代码的对应实现。

`lab06/6_2.asm`

- 用 `table dw prog0, prog1, prog2, prog3` 保存各功能入口标号。
- 菜单项包括：
  - `0.exit`：退出程序。
  - `1.prog1`：计算 `z = x + y` 并输出。
  - `2.prog2`：小写字母转大写，使用 `and al, 0dfh`。
  - `3.prog3`：大写字母转小写，使用 `or al, 20h`。
- `lab06/6_2.png` 保存了 DOSBox 中菜单选择、运行和退出截图。

### 实验七：循环程序设计

指导书要求掌握 `loop`、`loope`、`loopne` 等循环指令，理解 `cmp`/条件转移构成的循环结构，并用循环实现排序算法。

`lab07/7_1.asm` 使用 3 x 8 的成绩数组完成：

- 用二维数组 `score` 保存三个班的成绩。
- 用 `max` 保存每班 90 分及以上人数，用 `min` 保存每班 60 分以下人数。
- 用 `good` 保存所有优秀成绩，用 `fail` 保存所有不及格成绩。
- 通过外层班级循环和内层学生循环遍历成绩。
- 使用 `score[bx][si]` 的基址变址相对寻址访问二维数组元素。
- 对 `good` 和 `fail` 分别进行冒泡排序，按降序排列。
- 因为 `BP` 默认使用 `SS` 寻址，写入 `fail[bp]` 时显式使用 `ds:` 段超越前缀。

当前数据对应的调试结果：

```text
max  = 03 03 03
min  = 02 02 02
good = 64 63 61 60 5E 5D 5C 5B 5A
fail = 3B 3A 37 36 2D 2B
```

按十进制理解：

```text
good = 100, 99, 97, 96, 94, 93, 92, 91, 90
fail = 59, 58, 55, 54, 45, 43
```

`lab07/task.png` 是题目截图，`lab07/7_1.png` 和 `lab07/7.png` 是流程图，`lab07/result.png` 是 DEBUG 内存结果截图。

### 实验八：子程序设计与综合应用

`lab08/8.asm` 使用多个子程序完成 5 名学生信息的输入、排序和输出：

- `inName` 使用 `int 21h/0Ah` 缓冲输入姓名，每个姓名记录占 15 字节。
- `inScore` 使用 `int 21h/01h` 逐字符输入成绩，成绩的 ASCII 字符保存在 `scoreA`，数值保存在 `scoreN`。
- `rank` 数组保存学生原始序号，排序时与 `scoreN` 同步交换。
- `sort` 对 `scoreN` 做冒泡降序排序，同时维护 `rank`，避免移动姓名和成绩字符串。
- `disName` 和 `disScore` 根据排序后的 `rank` 查找原始姓名、成绩字符串并用 `int 21h/09h` 输出。

`lab08/8流程图.drawio` 是实验八流程图源文件，`lab08/第8次课流程图参考/` 保存了流程图参考图片。

## 作业五

`hw5` 目录整理了第五次作业中的条件分支、数组统计、字符串输入和位操作题目。

- `5_26.asm`：通过移位和进位标志判断奇偶。
- `5_27.asm`：按 `x` 的正负和零值分段计算 `y`。
- `5_28.asm`：按 `x` 的符号分段计算 `z`。
- `5_29.asm`：统计字数组中的正数、负数和零。
- `5_30.asm`：输入字符串，在空格或长度上限处结束。
- `5_31.asm`：提取寄存器中的特定位字段。

## 作业六

`hw6` 目录整理了第六次作业中的串操作、循环累加、数组查找、字符输出和排序题目。

- `6_14.asm`：使用 `rep movsb` 将数据段中的字符串复制到附加段。
- `6_15.asm`：循环计算 `1*2+3*4+5*6+7*8`，并将结果保存到变量 `x`。
- `6_16.asm`：在字符数组中查找 `@`，找到后将 `sign` 置为 1。
- `6_18.asm`：输入一个数字字符，循环输出从该数字开始的 10 个数字，超过 `9` 后回到 `0`。
- `6_20.asm`：遍历数字字符数组，找出最小字符并保存到 `min`。
- `6_22.asm`：对 30 名学生成绩进行冒泡排序，按升序排列。

## 作业七

`hw7` 目录整理了第七次作业中的子程序设计、数字转换、数组处理和位统计题目。部分文件是独立完整程序，部分文件只给出可被主程序调用的 `proc` 子程序，并在文件开头用注释说明需要的数据段变量或入口参数。

- `7_16.asm`：从键盘逐字符输入数字，转换为数值后顺序保存到 `buffer`，并用 `count` 记录输入位数。
- `7_17.asm`：完整程序，循环读入十进制数字字符，调用 `in_num` 子程序按 `x = x * 10 + digit` 累加。
- `7_18.asm`：`sum` 子程序，按 `array` 和 `num` 遍历字节数组，并将求和结果保存到 `result`。
- `7_20.asm`：`show` 子程序，将字节变量 `x` 的高、低两个十六进制数字依次输出。
- `7_21.asm`：`show` 子程序，将 `BX` 中的无符号二进制数转换为十进制字符输出。
- `7_28.asm`：`sum` 子程序，统计 `AX` 的 16 个二进制位中有多少位为 1，并将结果放入 `CL`。
- `7_29.asm`：`find_max` 子程序，遍历字节数组 `array`，把最大值保存到 `max`。
- `7_30.asm`：完整程序，调用通用 `sum` 子程序分别计算数组 `A`、`B` 中负数元素之和，并保存到 `SUM1`、`SUM2`。
